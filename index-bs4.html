Boostrap 4 scrollable tabs

Prevents Bootstrap tab controls from automatically wrapping when there is insufficient room to render all tab links on a single line.

Try resizing the browser window width down - eventually this will trigger scroll links to appear on either side of the control. Source code available at https://github.com/ablewhite/bootstrap-scrollable-tabs.
